# Single-Tree Inference Simulation Studies - File Manifest
# Created: 2026-04-29

## Purpose
Determine if single full-sample tree can be used with valid inference through:
1. Study 1: Testing similarity between full-sample and cross-fitted trees
2. Study 2: Testing coverage of bias-adjusted confidence intervals
