### Hypothesis: Variance Estimation Problem

Since nuisances improve, bias is low, and RMSE decreases with n, but coverage worsens, the most likely cause is **variance estimation formula σ² = mean(ψ²) systematically underestimates true variance**.

**Possible mechanisms:**
1. Score function not properly centered: mean(ψ) ≠ 0
2. Cross-fitting with K=5 induces dependence (only 4 covariate patterns with binary X1, X2)
3. Variance formula invalid for discrete covariates / 4-leaf trees
4. Sample splitting causes high variance with K=5

## Session: ATT Orthogonal Score Correction - Implementation Verification (Evening)

### Context

Verified and confirmed the implementation of the corrected ATT orthogonal score formula from Chernozhukov et al. (2018). This critical correctness fix was already implemented in commit 4e3c307 ("Sprint 2") earlier today.

### The Problem (Resolved)

**Previous (WRONG) implementation:**
- Used 3 nuisances: e(X), m0(X), m1(X)
- Score formula: `ψ = (A/π)(Y - m0 - θ) - (1/π)((A - e)/(1 - e))(m1 - m0)`
- Manuscript equation (2) had incorrect sign and structure

**Theoretically incorrect because:**
- ATT estimation only requires E[Y|A=0,X], not E[Y|A=1,X]
- Chernozhukov (2018) efficient influence function uses 2 nuisances
- Using m1 was unnecessary and violated theoretical basis

### The Fix (Implemented in Sprint 2)

**Corrected implementation:**
- Uses 2 nuisances: e(X), m0(X) only
- Correct Chernozhukov (2018) formula:
  ```
  ψ = (A/π)(Y - μ₀(X) - θ) - (1/π) · (e(X)(1-A)/(1-e(X))) · (Y - μ₀(X))
  ```

**Key features of correct formula:**
- MINUS sign (subtraction) in second term
- Second term: (1/π) not ((1-A)/(1-π))
- Second term numerator: e(X)(1-A)
- Only requires control outcome regression m0(X) = E[Y|A=0,X]
